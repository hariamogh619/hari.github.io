Navigation

Navigation is a component that enables visitors to move from one page to another. It states the overall information architecture of the site. It is present on all the pages of the portfolio. It also reveals the brand identity using through text or logo.

Desktop and Tablet view

Mobile view

Best practices & Accessibility

  1. Use Nav tag to attribute navigation to assistive technologies.
  2. Add role='navigation' to attribute navigation to assistive technologies.
  3. Always have a branding element attached to the nav bar to portray the purpose of the site.
  4. Have only the important menus in the nav bar.
